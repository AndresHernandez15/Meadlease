# Módulo HMI (Interfaz Humano-Máquina)

**Estado:** 🟢 En desarrollo

Este directorio contiene la implementación del HMI para el robot Meadlese.

## Arquitectura

La arquitectura final se basa en un stack **FastAPI + WebSocket + Chromium Kiosk**. La documentación técnica detallada, incluyendo la estructura de archivos de destino y el mapa de estados, se encuentra en:

- **[📄 Ver Documentación Completa del HMI](../docs/DOCUMENTACION_HMI.md)**

## Prototipos Actuales

Los archivos en la raíz de este directorio (`index_test.html`, `baymax_face.js`) son **prototipos iniciales** utilizados para validar el concepto de la cara animada. Serán integrados en la estructura final `frontend/static/js/` como parte de la implementación completa.
